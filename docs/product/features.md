# DNF Mobile Action - 产品功能

## 完整游戏流程

### 账号登录系统（纯前端 localStorage）
- 支持注册/登录，密码加密存储在本机
- 自动记住登录状态，刷新页面无需重新登录
- 游客模式一键进入
- 支持键盘 Tab 切换输入框、Enter 提交

### 主菜单界面
- 角色立绘展示 + 属性面板（HP/MP/ATK）
- 技能预览列表
- 开始副本 / 练习模式 / 角色信息 / 退出登录
- 悬停高亮交互

### DNF 风格战斗界面
- 左下固定方向轮盘 + 十字辅助线
- 右下大攻击按钮 + 独立跳跃按钮
- 攻击按钮上方四个技能槽，带冷却遮罩动画
- 键盘：WASD移动，J攻击，K跳跃，Shift冲刺，Q/E/R/F技能

### 战斗系统
- 普通攻击 + 4技能：上挑(击飞)、地裂波(远程)、旋风斩(AOE)、崩山击(冲锋)
- 击中顿帧、屏幕震动、击退、刀光特效
- 冲刺带残影，跳跃重力模拟
- 连击评级 F~SSS 八级

### 波次副本制
- 5波递增难度，第5波Boss战
- Boss顶部血条、死亡爆炸特效
- 通关/失败结算面板

### 四种敌人 + Boss
- 哥布林/猫妖/牛头兵/铠甲兵 + 牛头王
- 巡逻/追击/攻击 AI 状态机
