# DNF Mobile Action - 产品功能

## 核心功能

### DNF 风格操作界面
- 左下角固定方向轮盘（不随画面移动），十字辅助线，带死区防误触
- 右下角大攻击按钮 + 独立跳跃按钮
- 攻击按钮上方四个技能槽（上挑/地裂波/旋风斩/崩山击），带冷却遮罩动画
- 键盘备用：WASD移动，J攻击，K跳跃，Shift冲刺，Q/E/R/F释放技能

### 完整战斗系统
- 普通攻击 + 4个冷却技能，每种技能不同效果（近战/投射物/AOE/冲锋）
- 击中顿帧（Hit Stop）增强打击感
- 冲刺系统带残影效果
- 跳跃支持，重力模拟
- 连击评级系统：F/D/C/B/A/S/SS/SSS

### 波次副本制
- 5波敌人递增，第5波为Boss战
- 波次之间自动过渡，清理当前区域后推进
- Boss血条显示在屏幕顶部
- 全清后显示通关界面

### 四种敌人类型
- 哥布林（基础）、猫妖（快速）、牛头兵（重型）、铠甲兵（精英）
- 牛头王 Boss（大血条、死亡爆炸效果）
- 巡逻/追击/攻击 AI 状态机

### HUD 信息面板
- 左上角角色面板：头像、等级、HP条（绿黄红）、MP条（蓝）
- 右上角：波次进度、击杀数
- 中央：连击评级（带颜色分级和弹跳动画）
- 底部：操作提示
