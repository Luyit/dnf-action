# DNF Mobile Action - 产品功能

## 核心功能

### 虚拟摇杆操作
- 屏幕左下角虚拟轮盘，支持触摸/鼠标拖拽控制
- 同时支持键盘 W/A/S/D 方向键操作
- 四方向移动，支持上下切换三条通道

### 横版战斗系统
- 点击/按J键触发攻击，带刀光特效
- 攻击命中判定，敌人受击反馈（击退+闪烁）
- 连击系统，5连击以上屏幕显示连击数

### 敌人AI系统
- 三种敌人类型：哥布林(基础)、牛头兵(重型)、猫妖(快速)
- 巡逻、追逐、攻击三状态AI切换
- 随游戏时间逐渐增加难度

### 战斗HUD
- 左上角血条（绿/黄/红三色变化）
- 右上角击杀数和分数统计
- 屏幕中央连击特效

### 横版卷轴地图
- 带城市剪影的三层视差背景
- 世界动态扩展，无边界限制
- 动态难度，随游戏时间加快敌人刷新
